# Electroproject — инженерные расчёты через локальный веб-сервер

Все расчёты запускаются через браузер на **http://127.0.0.1:8765**.

## Требования

- Python 3.10+
- Windows (для СОПТ + CDW нужен **КОМПАС**)
- Зависимости: `pip install -r requirements.txt`

## Запуск

```bash
python run_web.py
```

или `run_web.bat`. По умолчанию сервер доступен **и на этом ПК, и в локальной сети** (`0.0.0.0:8765`).

- На этом компьютере: **http://127.0.0.1:8765/**
- Для других ПК в сети: **http://<IP-этого-компьютера>:8765/** (адрес показывается в консоли при запуске)

Только локально (без сети): `python run_web.py --local`

При необходимости разрешите порт **8765** в брандмауэре Windows.

## Модули на главной странице

| Кнопка | Файлы | API |
|--------|-------|-----|
| Проверка на невозгорание | Excel `.xlsx` | `/api/fire_check/run` |
| Расчёт СОПТ | Excel + CDW | `/api/sopt/run` |
| Расчёт ЩСН | Excel модели сети | `/api/shsn/run` |
| Проверка оборудования | — | в разработке |

После расчёта Word-отчёт скачивается автоматически.

## Структура

```

├── app/
│   ├── web/                # FastAPI + страницы
│   └── calculations/       # логика расчётов

```
